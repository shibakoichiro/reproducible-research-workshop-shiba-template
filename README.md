# Reproducible Research Workshop

Welcome! This repository is your workspace for the **Reproducible Research Workflow Workshop**.

## Before the Workshop (Setup Check)

Complete these steps **before** the workshop to verify your environment is ready.

### Step 1: Clone this repository

In Positron (or your IDE):
1. Open the **Source Control** panel
2. Click **Clone Repository**
3. Paste the URL of this repository
4. Choose a location on your computer
5. Open the cloned folder

### Step 2: Complete the setup check

1. Open `setup-check.qmd`
2. Replace `"YOUR NAME"` with your name
3. Click **Render** (or Ctrl+Shift+K / Cmd+Shift+K)
4. Verify that the HTML output is created without errors

### Step 3: Commit and push

1. Go to **Source Control** panel
2. Stage the changes to `setup-check.qmd` (click **+** next to changed files)
3. Write a commit message: `Complete setup check`
4. Click **Commit**
5. Click **Push** (or Sync)

Note: The rendered HTML output (in `_manuscript/`) is ignored by Git. Only commit the source `.qmd` files.

If all steps complete without errors, you're ready for the workshop!

## During the Workshop

You will create and work with these files:

| File | Purpose |
|------|---------|
| `data-cleaning.qmd` | Load and clean NHANES data |
| `analysis.qmd` | Create tables and figures |
| `index.qmd` | Write your research manuscript |

### Pair Collaboration (Part 7)

You will collaborate with a partner:
1. Add your partner as a **Collaborator** (Settings → Collaborators)
2. Clone your partner's repository
3. Review and edit their `index.qmd`
4. Commit and push your changes
5. Pull changes to see what your partner added to your manuscript
